🎬 Watch Next
Watch Next, kullanıcıların en yeni ve popüler filmleri keşfedebileceği, detaylarını inceleyebileceği ve fragmanlarını izleyebileceği modern bir film platformudur. TMDB API altyapısını kullanan bu uygulama, React ve Vite ile yüksek performanslı bir kullanıcı deneyimi sunar.

🚀 Özellikler

* Popüler ve Güncel Filmler: TMDB API üzerinden anlık çekilen film listeleri.

* Film Detayları: Özet, puanlama, vizyon tarihi ve oyuncu kadrosu bilgileri.

* Fragman İzleme: YouTube entegrasyonu ile filmlerin fragmanlarına kolay erişim.

* Favorilere Ekleme: Beğendiğiniz filmleri daha sonra göz atmak için listenize ekleme özelliği.

* Hızlı ve Modern Arayüz: Vite ve React ile optimize edilmiş, akıcı bir deneyim.

🛠️ Kullanılan Teknolojiler

* Frontend: React.js

* Build Tool: Vite

* API: The Movie Database (TMDB)

* Styling: CSS3

📦 Kurulum

Projeyi yerel makinenizde çalıştırmak için şu adımları izleyebilirsiniz:

Depoyu klonlayın:

```
git clone https://github.com/UmutCnAltun/watch-next.git
```
Proje dizinine gidin:

```
cd watch-next
```

Bağımlılıkları yükleyin:
```
npm install
```
API Anahtarınızı Ayarlayın:
tmdb.js dosyasına girin ve TMDB API anahtarınızı ekleyin:
```
const API_KEY = 'ENTER_YOUR_KEY_HERE';
```
Uygulamayı başlatın:

```
npm run dev
```
